<h2>Project Overview</h2>
</div>

- **Role:** Senior Product Designer
- **Company:** Thermondo
- **Outcome:** 52% more leads generated from the new Multi-step Calculator.

Thermondo is the leading heating installer in the field of single and double family homes in Germany. Between 2013-2015 it grew by 650%. With the digitalization of the heating exchange, Thermondo shapes the future's handiwork.

<div class="detail-section-header">
<h2>Product Challenge</h2>
</div>

- Apply a user centered design approach
- Improve the look and feel especially for the homepage
- Improve responsive mobile and desktop UX
- Define a complete new navigation for the growing content
- Make it easier for marketing to create new sites without the help of the frontend team (reusable elements)
- Keep stakeholders happy :-)

<div class="detail-section-header">
<h2>What did I work on?</h2>
</div>

### Structure
- Planned and created the structure (user flows, sitemap, navigation) based on the new needs of all stakeholders.
- Simplified margins for all site elements to one margin size, so content creators can use them in various orders when they create new pages.
- Simplified site elements (like Images, text fields, banners etc.) which are now easy to maintain, no more reinventing the wheel for each new page.

### Interface Design
- Collected needs for the website from different stakeholders.
- Won over a difficult stakeholder by designing detailed wireframes for every screen and state.
- Build wireframes for all pages and testing them.
- Designed all the pages, icons and the color guide.
- Created an editable styleguide.

<h2>The Old Multistep Calculator</h2>
</div>

### Before Redesign
This is how the process of generating an offer was designed before I joined the company.

**Issues to solve:**
- Distracting design.
- Lack of a clear user flow for visitors from different sources, such as ads or newsletters.
- Excessive number of tabs in the progress bar, causing users to wonder how long the process will take.
- The progress bar itself doesn't fit visually to the content - it reaches far to the right, but the clickable icons and the CTA are on the left.
- The key elements (Erdgas, Heiz√∂l, Fl√ºssiggas, Nachtspeicher/strom) to click have a confusing design out of 3 elements: icon, text and circle input field.
- The background image of the house on the left get's ignored by most of the people.
- The photo looks like stock photography and don't add any trustworthy value.
- Too many elements below the multistep calculator distracting the user from the main content.
- Missing a good mobile/tablet version.

<div class="detail-section-header">
<h2>Multistep Calculator ‚Äî After My Redesign</h2>
</div>

### After Redesign
Some changes are done!

- Clean design.
- Same header (initially), because I wanted to test the impact of the calculator itself without influencing the conversion rate.
- Later A/B tested photos of one of Thermondo's real Heizungsbauer and new, better working copy.
- The progress navigation has only 4 points and looks as part of the whole multistep.
- The key elements which should be used by the user are now the main focus of the page and now they look more like clickable buttons.
- Elements which had no big influence on the conversation rate or shouldn't be part of the first page of the multistep at all got removed.
- Full responsive support for mobile, tablet, and desktop.
